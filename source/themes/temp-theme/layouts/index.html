What?

Temp monitor is an open source and hardware monitor to show system resources such as CPU and GPU temperature, fan speeds, RAM usage, GPU and CPU load.

Why?

There are similary products on the market such as fan speed controllers, but these usualy only show air temperature and fan speeds and not the actual temperatures of the components or any other information. Temp monitor instead shows the actual temperatures and information collected by the computer. This provide an easy way of monitor you computer while playing or using it as an server and can help optimizing computer performence, health and maximizing computer lifetime.

Temp monitor comes with an open license meaning that if you feel that any function is missing or should be change you can take part in the design and programing.

How?

Temp monitor is simple to use and plugs in with a micro-USB cable to your computer. Install our simple software and you're ready to go. When installed and plugged in the computer software collects information and sends it to the monitor.
